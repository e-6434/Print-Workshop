# 🖨️ Print Workshop Management System

A modern desktop-based management system for print workshops, built with **React**, **Express**, and **SQLite**.

> 🚧 This project is currently under active development.

---

## ✨ Features

- 📋 Customer Management
- 🧾 Order Management
- 🖨️ Print Job Tracking
- 📦 Product Management
- 👥 User Authentication
- 📊 Dashboard & Reports
- ⚙️ Settings Management

---

## 🛠️ Tech Stack

| Technology | Description |
|------------|-------------|
| React + Vite | Frontend |
| Express.js | Backend API |
| SQLite | Database |
| Git & GitHub | Version Control |
| Electron | Desktop Packaging (Final Release) |

## 🚀 Getting Started

### Clone

```bash
git clone https://github.com/e-6434/Print-Workshop.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
